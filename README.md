# FilePilot
A fast and lightweight desktop file &amp; folder search application built with Python and Tkinter.

Smart File & Folder Search Tool

SwiftSeek is a modern desktop search utility developed using Python and Tkinter that allows users to quickly search files and folders across selected drives on their computer.

The application provides a clean graphical interface, multithreaded searching for better responsiveness, dynamic drive detection, and real-time search status updates.

Designed for simplicity, speed, and usability.

✨ Features
🔍 Search files by name
📁 Search folders by name
💽 Dynamic drive detection
✅ Multi-drive selection
⚡ Multithreaded search system
⏹ Stop search anytime
🧹 Clear results instantly
📜 Scrollable results panel
🎨 Clean and user-friendly GUI
📊 Search time tracking
🖥 Lightweight desktop application
🛠 Technologies Used
Python
Tkinter

OS Module
